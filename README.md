# 🚀 AI & Quantitative Engineering Portfolio

歡迎來到我的個人網站與作品集倉庫。本專案採用**完全解耦（Decoupled）**的架構設計，將網頁結構（HTML）、視覺樣式（CSS）與核心資料層（JavaScript Data Store）徹底抽離，旨在打造一個兼具極致視覺互動、高動態擴充性，且對 SEO 友善的專業工程師動態履歷網站。

## 視覺設計

- **資料與視圖徹底解耦**：所有個人資訊、專業技能、精選專案、職涯經歷皆模組化封裝於 `data.js` 中。未來新增作品或修改描述時，**無需改動任何 HTML 標籤或 Tailwind 語法**，實現資料驅動（Data-driven）的輕量級網頁渲染。
- **高階 3D 懸浮互動特效**：專業技能區塊採用原生 JavaScript 實現高性能的滑鼠軌跡監聽，動態計算相對座標並即時注入 CSS 變數，達成平滑的 **3D 視角立體傾斜** 與 **動態 Spotlight 局部光暈發光** 特效。
- **針對 Wan2.2 比例視覺優化**：專案展示區塊針對 AIGC 最新影片生成大模型 Wan2.1 的 **768×1024 縱向長型比例 (3:4)** 進行排版自適應優化，完美防止多媒體畫面裁切，並採用 60%:40% 的黃金雙欄分割比例。
- **多端自適應響應式佈局 (RWD)**：全站基於 Tailwind CSS 進行原子化樣式建構，在 4K 螢幕、筆記型電腦乃至行動端設備上均具備極佳的排版流暢度。
